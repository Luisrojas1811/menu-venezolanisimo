# Menú Digital - Maf Sazón 🥘

Proyecto desarrollado para digitalizar la carta de un comercio local, optimizando la experiencia del cliente mediante un sistema interactivo y ligero.

## 🛠️ Tecnologías utilizadas
* **Frontend:** HTML5 semántico y CSS3 (Layouts con Flexbox/Grid).
* **Lógica:** JavaScript vanilla (Manipulación dinámica del DOM).
* **Estado:** Proyecto totalmente funcional y optimizado para dispositivos móviles.

## 🚀 Características principales
* **Velocidad de carga:** Al no depender de frameworks pesados, el sitio carga instantáneamente en cualquier celular.
* **Interfaz intuitiva:** Diseño pensado para que el usuario encuentre los productos de forma rápida.
* **Escalabilidad:** Código organizado para facilitar la transición a un framework como React en el futuro.

## 📌 Objetivo del proyecto
Este sistema fue creado para resolver una necesidad real de un negocio, aplicando las buenas prácticas de desarrollo que vengo incorporando en la **UADE**.
